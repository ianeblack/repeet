# INSTALLATION

### Download the repository from github into your account

```
git clone https://github.com/ianeblack/repeet.git
```

```
cd repeet
```

```
chmod +x repeet
```

```
sudo cp repeet ~/../../usr/local/bin/repeet
```

# USAGE

`repeet -s "balls" -r 90`
